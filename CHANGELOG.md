# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2026-02-05

### ✨ Added
- **객체지향 JavaScript 구조**: 클래스 기반 모듈화된 코드 아키텍처
- **프리미엄 디자인 시스템**: 고급스러운 색상 팔레트와 타이포그래피
- **글래스모피즘 효과**: 반투명 배경과 블러 효과 적용
- **스크롤 애니메이션**: Intersection Observer를 활용한 페이드인 효과
- **CSS 변수 시스템**: 일관된 디자인을 위한 커스텀 프로퍼티
- **유틸리티 클래스**: 재사용 가능한 CSS 클래스들
- **접근성 향상**: ARIA 레이블 및 키보드 네비게이션 개선
- **상품 카테고리**: 각 상품에 카테고리 속성 추가
- **README.md**: 상세한 프로젝트 문서
- **package.json**: NPM 패키지 설정
- **.gitignore**: Git 버전 관리 설정

### 🎨 Changed
- **색상 시스템**: 더 깊고 프리미엄한 다크 테마
- **타이포그래피**: Inter 폰트 적용 및 크기 조정
- **간격 시스템**: 일관된 spacing 변수 사용
- **버튼 디자인**: 리플 효과 및 호버 애니메이션 개선
- **카드 디자인**: 그림자와 그라데이션 효과 강화
- **모달 디자인**: 더 큰 크기와 부드러운 애니메이션
- **헤더**: 고정형 헤더에 블러 효과 추가

### 🚀 Improved
- **성능 최적화**: will-change 속성 활용
- **애니메이션 성능**: GPU 가속 최적화
- **코드 구조**: 관심사 분리 및 모듈화
- **가독성**: 코드 주석 및 문서화 개선
- **반응형 디자인**: 모바일 경험 향상
- **로딩 속도**: 폰트 프리로드 및 이미지 최적화

### 🔧 Fixed
- **CSS 호환성**: 브라우저 호환성 개선
- **접근성 문제**: 스크린 리더 지원 강화
- **모바일 레이아웃**: 작은 화면에서의 표시 문제 해결
- **애니메이션 버그**: 중복 애니메이션 제거

### 🎯 Refactored
- **JavaScript**: 클래스 기반 구조로 전면 리팩토링
  - `NexiShopApp`: 메인 애플리케이션 클래스
  - `AnimationManager`: 애니메이션 관리
  - `ProductManager`: 상품 렌더링 및 모달 관리
  - `ModalManager`: 모달 이벤트 처리
  - `AccordionManager`: 아코디언 기능
  - `LinkManager`: 링크 및 스크롤 관리
  - `FooterManager`: 푸터 업데이트
- **CSS**: 섹션별 주석 및 구조화
- **HTML**: 시맨틱 마크업 개선

## Version Naming

- **Major (X.0.0)**: 대규모 구조 변경, 호환성 깨짐
- **Minor (0.X.0)**: 새로운 기능 추가
- **Patch (0.0.X)**: 버그 수정 및 소규모 개선
